YelpCamp

YelpCamp is a full web service using Bootstrap, EJS, Node.js, Express.js, and mongoDB. Deployed to Heroku with Git and database hosted by AWS using mongoDB Atlas. Features dynamically loaded databased content, RESTful routes, user authentication and authorization.

About Me

My journey in programming started with the University of Helsinki’s massive open online course (MOOC), Object Oriented Programming with Java. From there, I took Colt Steele's Udemy course, The Web Developer Bootcamp. During and after which, I made this website and all the projects it contains. Code-alongs are great for learning, but to develop your skills you have to use them, so personal projects have been my bread and butter, filling in the gaps and expanding on what these courses have taught me.
